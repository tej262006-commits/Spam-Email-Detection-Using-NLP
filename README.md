# Spam Email Detection Using NLP

## Project Overview
This project detects whether a message is Spam or Ham using Natural Language Processing (NLP) and Machine Learning.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Machine Learning Algorithm
- Multinomial Naive Bayes

## Features
- Text preprocessing
- TF-IDF Vectorization
- Spam/Ham prediction
- Model accuracy evaluation

## Dataset
SMS Spam Collection Dataset (5572 messages)

## Result
The model achieves high accuracy in detecting spam messages.
